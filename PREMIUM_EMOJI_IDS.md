# Премиум Эмодзи для Telegram ботов

## Для сообщений (используй `<tg-emoji emoji-id="...">эмодзи</tg-emoji>`)

```python
# Пример в сообщении:
text = f'<tg-emoji emoji-id="5904462880941545555">🪙</tg-emoji> Ваш баланс: {bal} монет'
await msg.answer(text, parse_mode="HTML")
```

## Для инлайн кнопок (используй `icon_custom_emoji_id="..."`)

```python
# Пример в кнопке:
InlineKeyboardButton(
    text="Баланс",
    callback_data="balance",
    icon_custom_emoji_id="5904462880941545555"  # 🪙
)
```

## Примеры кода

### Пример 1: Меню с премиум кнопками

```python
def admin_menu_kb():
    return InlineKeyboardMarkup(inline_keyboard=[
        [InlineKeyboardButton(
            text="Список пользователей",
            callback_data="admin_users",
            icon_custom_emoji_id="5870772616305839506"  # 👥
        )],
        [InlineKeyboardButton(
            text="Рассылка сообщения",
            callback_data="admin_broadcast",
            icon_custom_emoji_id="6039422865189638057"  # 📣
        )],
    ])
```

### Пример 2: Сообщение с премиум эмодзи

```python
@dp.message(Command("admin"))
async def cmd_admin(msg: Message):
    text = (
        f'<tg-emoji emoji-id="5870982283724328568">⚙️</tg-emoji> <b>Админ-панель</b>\n\n'
        f'<tg-emoji emoji-id="6039422865189638057">📣</tg-emoji> Управление ботом'
    )
    await msg.answer(text, parse_mode="HTML", reply_markup=admin_menu_kb())
```
